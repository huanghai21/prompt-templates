# 学习笔记生成

## 提示词

```
请帮我将以下{学科/主题}的学习材料转化为结构清晰的学习笔记。笔记应包含：

1. 核心概念和定义（用简明语言解释）
2. 关键要点（使用要点符号列表）
3. 概念之间的关系或框架图（用文字描述可视化结构）
4. 实例或应用场景
5. 易混淆点或难点解析
6. 记忆技巧或助记方法

学习材料：
{粘贴学习材料内容}
```

## 示例

```
请帮我将以下心理学的学习材料转化为结构清晰的学习笔记。笔记应包含：

1. 核心概念和定义（用简明语言解释）
2. 关键要点（使用要点符号列表）
3. 概念之间的关系或框架图（用文字描述可视化结构）
4. 实例或应用场景
5. 易混淆点或难点解析
6. 记忆技巧或助记方法

学习材料：
认知偏差是指人类在处理信息时系统性偏离规范或理性判断的思维模式。确认偏差描述了人们倾向于寻找、诠释、偏爱和回忆与已有信念相符的信息，同时给予与这些已有信念相矛盾的信息较少的重视或考虑。这种偏差在信息丰富但注意力有限的环境中尤为明显，个体往往会使用各种启发式方法来处理信息。

锚定效应是认知偏差的一种，指人们在做决策时过分依赖最初获得的信息（称为"锚"）。例如，在价格谈判中，最初提出的价格会显著影响最终达成的协议。框架效应则描述了信息呈现方式对决策的影响—同样的信息以不同方式呈现会导致不同的决策结果。

可用性启发式是指人们根据能够想到的例子的容易程度来判断事件的可能性。这种倾向在评估风险时尤为重要。群体思维是指团体中的个体为了和谐一致而抑制异议，可能导致非理性决策。

理解认知偏差对个人决策、组织行为和社会心理学有重要意义。通过意识到这些偏差，人们可以采取策略减少其影响，如寻求反面证据、考虑多种解释和使用结构化决策过程。
```

## 说明

这个提示词用于将冗长或复杂的学习材料转化为结构化的学习笔记，便于理解和记忆。适用于各类学科的学习辅助。
